# Image Resizer Tool

This project resizes and converts all images inside a folder using Python and Pillow.

## Tools Used
Python
Pillow Library

## Features
Resize images in batch
Convert JPG to PNG
Automatic folder reading

## How to Run
1. Install Pillow
pip install pillow

2. Add images in input_images folder

3. Run script
python resizer.py

4. Resized images will be saved in output_images folder
